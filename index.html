Welcome to the GitHub repository for Minoggin, a work-in-progress I began in July 2024.

At the moment, this repository serves as both a place to manage version control and a sneak peek at my evolving website. You'll find educational content already available, with much more to come as I continue building. Since it's a work in progress, expect to see some unfinished parts and rough edges while I develop the final version for the full launch.
